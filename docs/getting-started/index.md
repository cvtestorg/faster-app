## 🚀 快速开始

如果你已经熟悉 FastAPI 和 Python, 可以快速开始:

```bash
# 1. 安装
uv add faster-app

# 2. 创建项目
uv init my-project && cd my-project
faster app demo

# 3. 启动服务器
faster server start
```

✅ **完成!** 访问 http://localhost:8000 查看你的应用
